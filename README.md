# QR Code Generator

A modern, feature-rich QR code generator built with React, TypeScript, and Tailwind CSS.

## Features

- Generate QR codes from text, URLs, or contact information
- Customize QR code colors, style, and corner design
- Add a custom logo to your QR code
- Download QR codes in PNG, SVG, or JPEG formats
- Light and dark mode support
- Responsive design for mobile and desktop

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Vite
- qrcode.react library

## Getting Started

### Prerequisites

- Node.js (v14.0.0 or later)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Subhash-04/QR-Generator.git
   cd QR-Generator
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Building for Production

```bash
npm run build
# or
yarn build
```
